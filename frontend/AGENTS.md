# Scope: frontend local app

Usa este directorio para la UI React/Vite de la app local.

## Reglas
- Lee `README.md` y `../README.md` solo cuando necesites detalles de arranque o build.
- Manten alineados los contratos con `app_backend/` y los endpoints `/api/intelligence/*`.
- Prefiere flujos de datos servidos por backend; no metas lecturas directas del browser sobre `output/`.
- Conserva el lenguaje visual ya existente salvo que el usuario pida un rediseño explicito.
- Si cambias logica de UI o tipos, valida con el build del frontend.
