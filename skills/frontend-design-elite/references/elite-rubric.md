# Elite Rubric

Puntuar cada dimension de 1 a 10. Objetivo minimo: promedio >= 8 y ninguna dimension < 7.

## 1. Direccion visual
- 9-10: identidad clara, memorable, coherente de extremo a extremo.
- 7-8: buena direccion pero con algunos bloques genericos.
- <=6: look comun sin sello propio.

## 2. Jerarquia y composicion
- 9-10: lectura inmediata, foco visual correcto, ritmo entre secciones.
- 7-8: estructura correcta con pequeños problemas de prioridad.
- <=6: competicion visual, escaneo dificil.

## 3. Sistema de diseno
- 9-10: tokens consistentes, escalas claras, estados completos.
- 7-8: base correcta con huecos menores de consistencia.
- <=6: estilos ad-hoc sin sistema.

## 4. Motion y feedback
- 9-10: animaciones utiles, suaves, con proposito funcional.
- 7-8: motion correcto pero poco expresivo o repetitivo.
- <=6: animaciones ausentes o decorativas sin valor.

## 5. Responsive y accesibilidad
- 9-10: mobile y desktop solidos, foco visible, contraste robusto.
- 7-8: buena base con ajustes pendientes menores.
- <=6: roturas, baja legibilidad o interaccion deficiente.

## 6. Calidad de implementacion
- 9-10: codigo limpio, mantenible, semantico y facil de extender.
- 7-8: implementacion funcional con deuda tecnica moderada.
- <=6: codigo fragil o dificil de mantener.

## Criterios de bloqueo (no entregar)
- Uso de layout/paleta claramente generica sin direccion de arte.
- Tipografia default sin justificacion de sistema existente.
- Falta de validacion mobile.
