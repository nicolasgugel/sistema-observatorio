# Art Directions

Seleccionar una sola direccion principal por entrega. Evitar mezclar 3-4 estilos sin criterio.

## 1. Editorial Bold
- Mejor para: landing pages de marca, storytelling, lanzamientos.
- Tipos sugeridas: `Bricolage Grotesque` + `IBM Plex Sans`.
- Paleta base: carbon `#101114`, marfil `#f4f1ea`, acento cobre `#c97a3d`.
- Claves: titulares grandes, composicion asimetrica, fotografias o bloques con recorte.

## 2. Precision Fintech
- Mejor para: dashboards, pricing, producto B2B.
- Tipos sugeridas: `Sora` + `Manrope`.
- Paleta base: tinta `#0d1321`, acero `#3e5c76`, hielo `#f0f4f8`, acento lima `#7ae582`.
- Claves: grid clara, densidad controlada, estados de datos muy legibles.

## 3. Tech Minimal Warm
- Mejor para: SaaS modernos con tono cercano.
- Tipos sugeridas: `General Sans` + `DM Sans`.
- Paleta base: tinta `#151515`, arena `#f8f4ec`, coral `#ff6b57`, azul polvo `#5c7cfa`.
- Claves: espacios amplios, bordes suaves, microdetalles de color en CTA y badges.

## 4. Neo Brutalist Product
- Mejor para: productos que buscan personalidad fuerte y recordacion.
- Tipos sugeridas: `Space Grotesk` + `Archivo`.
- Paleta base: negro `#111111`, blanco `#fdfdfd`, primario `#00d1ff`, secundario `#ffd60a`.
- Claves: contrastes altos, bordes marcados, bloques contundentes, iconografia simple.

## Seleccion rapida
- Si prima confianza y claridad de negocio: usar `Precision Fintech`.
- Si prima diferenciacion de marca: usar `Editorial Bold` o `Neo Brutalist Product`.
- Si prima equilibrio visual + conversion: usar `Tech Minimal Warm`.
